# Investigación - Patrones Arquitectónicos

## Objetivos de aprendizaje

* Identificar el uso de patrones arquitectónicos en soluciones de ingeniería de software (IV)

* Resolver un problema de ingeniería de software utilizando un patrón arquitectónico (VI)

## Especificación 

### Objetivo general

Preparar y presentar una exposición técnica sobre un patrón arquitectónico asignado, acompañada de una maqueta/prototipo de código que ejemplifique su implementación.

### Instrucciones

A cada grupo se le asignará un patrón arquitectónico a investigar. La investigación requiere realizar las siguientes actividades:

1. Análisis del patrón arquitectónico
   * Descripción general.
   * Problema que resuelve en términos de requerimientos no funcionales.
   * Limitaciones o desventajas.
   * Casos de uso en los que se recomienda aplicarlo.

2. Prototipo o maqueta de código
   * Incluye un diagrama modelo C4 de los primeros tres niveles (Contexto, Contenedores y Componentes).
   * Implementación en código (Java + SpringBoot) de los elementos programables principales del patrón y simulación (*mock*) de los componentes externos (bases de datos, sistemas, colas, entre otros).
   * Implementar un requerimiento funcional de ejemplo relacionado con un problema concreto para el cual este patrón es aplicable.
   * El prototipo no tiene que ser un sistema completo.

3. Exposición oral
   * Incluye los resultados del análisis del patrón arquitectónico y una explicación del prototipo.
   * No debe tomar más de 15 minutos.

## Rúbrica

Se evaluará la exposición oral con los siguientes rubros:

1. Claridad y estructura de la exposición (20 pts): fluida, bien organizada con un hilo conductor claro y una correcta administración del tiempo.
2. Dominio conceptual (20 pts): presenta contenidos correctos con precisión conceptual.
3. Correspondencia del prototipo (20 pts): el prototipo desarrollado se corresponde con el patrón arquitectónico.
4. Adecuada selección de problema concreto (20 pts): la aplicación del patrón arquitectónico es relevante para el problema concreto seleccionado.
5. Participación (20 pts): participa en las exposiciones de los otros grupos.
